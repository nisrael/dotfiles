# Nightfox Themes for Fuzzel

All nightfox theme variants are available for fuzzel. To switch themes, edit the `include` line in `fuzzel.ini`:

## Available Themes

### Dark Themes
- **carbonfox** (currently active) - IBM Carbon-inspired dark theme
  ```ini
  include=~/.config/fuzzel/carbonfox.ini
  ```

- **nightfox** - Default nightfox dark theme
  ```ini
  include=~/.config/fuzzel/nightfox.ini
  ```

- **nordfox** - Nord-inspired dark theme
  ```ini
  include=~/.config/fuzzel/nordfox.ini
  ```

- **duskfox** - Purple-tinted dark theme
  ```ini
  include=~/.config/fuzzel/duskfox.ini
  ```

- **terafox** - Teal/cyan dark theme
  ```ini
  include=~/.config/fuzzel/terafox.ini
  ```

### Light Themes
- **dawnfox** - Soft light theme
  ```ini
  include=~/.config/fuzzel/dawnfox.ini
  ```

- **dayfox** - High contrast light theme
  ```ini
  include=~/.config/fuzzel/dayfox.ini
  ```

## Switching Themes

1. Edit `~/.config/fuzzel/fuzzel.ini`
2. Change the `include` line to your preferred theme
3. The theme will apply immediately to new fuzzel windows
